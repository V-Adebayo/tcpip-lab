# IP Address

## What is an IP Address?

An IP (Internet Protocol) Address is a numerical identifier assigned to a device on a network. It allows devices to identify and communicate with each other.

## Types of IP Address

### IPv4

* Older version of IP addressing.
* Contains 32 bits.
* Written as 4 octets separated by dots.
* Each octet ranges from 0 to 255.

**Example:**
192.168.1.1

### IPv6

* Newer version of IP addressing.
* Contains 128 bits.
* Written in hexadecimal format.
* Developed to provide more available IP addresses.

**Example:**
2001:db8::1

## Static and Dynamic IP Addresses

### Static IP Address

* Does not change.
* Usually assigned manually or by an ISP.
* Commonly used for servers and network devices.

### Dynamic IP Address

* Can change over time.
* Usually assigned automatically by a router using DHCP.
* Commonly used in home and office networks.

## Public and Private IP Addresses

### Public IP Address

* Used to communicate over the Internet.
* Assigned by an ISP.

### Private IP Address

* Used within a local network (LAN).
* Not directly accessible from the Internet.

## How IP Addressing Works

```text
Device → Router → ISP → Internet
```

1. A device receives a private IP address from the router.
2. The router connects to the ISP.
3. The ISP provides a public IP address.
4. The public IP address is used to communicate over the Internet.

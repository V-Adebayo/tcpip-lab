This repository contains my hands-on practice and investigations of TCP/IP networking concepts using real tools like Wireshark, ping, and nslookup.

- TCP vs UDP
- TCP Three-Way Handshake
- Ports
- NAT

## Tools Used
- Wireshark
- Command Prompt / Terminal
- ping
- tracert
- nslookup

## Projects
- TCP Handshake Analysis
- Network Connectivity Investigation
- Port Analysis
- TCP vs UDP Traffic Analysis
- NAT Investigation

## Purpose
To build practical networking and cybersecurity skills through real packet analysis and network troubleshooting.
